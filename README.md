## MHSADTI
an end-to-end deep learning-based method named MHSADTI is developed to predict DTIs using multi-head self-attention and graph attention network.

## Requirements
- python >= 3.5
- torch >= 1.4.0
- RDkit >= 2019.03.30
- gensim >= 3.4.0
- numpy >= 1.16.1
- pandas >= 1.1.4
- transformers >= 3.1.0

## Using
1. main.py: start file for model training
